# Changelog Generator Skill Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Create a new skill directory `.agents/skills/changelog-generator/` containing the `SKILL.md`, analysis and verification scripts, and an example output file.

**Architecture:** The skill follows the project's modular skill pattern. It uses local bash scripts (`analyze_commits.sh`, `verify_changelog.sh`) to perform git history parsing and validation, ensuring deterministic output and avoiding hallucinations.

**Tech Stack:** Git, Bash, Markdown, JSON.

## User Review Required

> [!IMPORTANT]
